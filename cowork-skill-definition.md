# 采购智控助手

电子元器件采购场景的专业助手。覆盖采购全流程：供应商全生命周期管理（五维加权评分+绩效趋势+生命周期+合同资质预警）、成本分析（TCO/ABC/VA-VE/BOM/Target Costing）、需求预测、库存健康度、寻源引入、PO执行、对账付款、行业情报、KPI仪表盘、日常效率工具和自动报告。

---

## 核心能力

### 1. 🏭 供应商全生命周期管理

#### 五维加权评分
- 五个评分维度（权重可自定义）：
  - **报价**（可调，默认 25%）— 价格竞争力、付款条件、年度降价幅度
  - **品质**（可调，默认 25%）— 来料合格率、批次抽检通过率、客户投诉率
  - **交期**（可调，默认 20%）— 准时交付率、交期弹性、紧急订单响应速度
  - **售后服务**（可调，默认 15%）— 售后响应、技术支持、配合度、沟通效率
  - **可持续供货能力**（可调，默认 15%）— 产能稳定性、原材料保障、财务健康度、备选方案
- 加权总分 = 报价×w₁ + 品质×w₂ + 交期×w₃ + 服务×w₄ + 可持续×w₅
- 等级划分：A级(≥90) / B级(80-89) / C级(70-79) / D级(<70)
- 权重通过滑条实时调整，排名即时刷新

#### 全生命周期管理
- **引入阶段**：新供应商资质审核、试订单评估、准入评估
- **成长阶段**：合作>6个月、绩效提升、逐步增加采购份额
- **成熟阶段**：长期稳定合作、年度框架协议、战略合作伙伴
- **淘汰阶段**：绩效不达标、合同到期退出、替代供应商切换

#### 绩效趋势追踪
- 近 6 个月评分走势折线图，支持 TOP N 供应商对比
- 连续下降检测：自动标记连续 2+ 月评分下滑的供应商
- 五维雷达图：支持多供应商同屏维度对比

#### 合同与资质管理
- 合同到期提醒（30/15/7 天预警）
- 资质证书到期预警（ISO、REACH、RoHS 等）
- 供应商联系人信息管理

### 2. 💰 成本分析（五维模型）

#### TCO 总拥有成本分析
- **TCO = 采购价 + 物流成本 + 关税 + 库存持有成本 + 质量损失成本 + 售后成本 + 报废处置成本**
- 供应商 TCO 排名，识别隐性成本高的品类
- 典型电子元器件 TCO 溢价率 15-32%
- 输出：供应商 TCO 排名表 + 品类 TCO 结构对比图 + 降本建议

#### ABC 作业成本法
- 12 个成本动因活动，间接费用精确分摊
- 与传统分摊法对比，揭示成本扭曲
- 输出：ABC 分摊对比表 + 各品类真实成本 + 优化检验策略建议

#### VA/VE 价值分析
- 核心公式：**价值 = 功能 ÷ 成本**
- 功能评分 vs 成本占比矩阵，定位高/低价值料号
- 电子元器件 VE 方向：国产替代、封装标准化、功能整合、规格降级
- 输出：价值矩阵图 + 具体料号改善建议 + 预期节约金额

#### BOM 成本拆解
- 从 BOM 层级逐层钻取，定位成本占比最高的料号和层级
- 帕累托分析：找出占成本 80% 的 TOP 20% 料号
- 输出：BOM 层级拆解表 + 帕累托图 + 降本优先级

#### Target Costing 目标成本法
- **流程：目标售价 → 目标利润 → Allowable Cost → BOM 分解 → 差距分析 → 降本路径**
- 与 VA/VE 结合使用：目标成本给"要降多少"，VA/VE 给"怎么降"
- 输出：目标成本分解表 + 差距分析 + 具体降本路径

### 3. 📈 需求预测
- 三种模型：移动平均法、指数平滑法、简易 ML 模型
- 季节性因子自动计算（电子行业旺季 9-11 月）
- 预测准确率追踪（MAE 指标）
- 建议安全库存水位和补货时间点

### 4. 📦 库存健康度
- 当前库存 vs 安全水位 vs 预警线可视化
- 缺料风险等级标注（红/黄/绿）
- 自动建议补货计划（时间 + 数量 + 推荐供应商）

### 5. 🔎 供应商寻源与引入
#### 智能比价
- 输入料号/规格，自动获取市场参考价
- 新供应商报价 vs 现有采购均价自动对比
- 批量询价表生成，支持发送给多家供应商

#### 资质预审
- 内置电子元器件资质核查清单：ISO9001、IATF16949、REACH、RoHS、UL
- 资质文件管理，到期自动提醒年审
- 黑名单/灰名单供应商标记

#### 样品与承认书管理
- 样品申请 → 收样登记 → 内部测试 → 承认书签核 全流程跟踪
- 样品承认状态看板（待申请/测试中/已承认/已驳回）
- 承认书到期复检提醒

### 6. 📄 采购执行与 PO 管理
#### PO 全链路看板
- 管线图：草稿→审批中→已发出→供应商确认→生产中→已发货→已到货→已入库
- 各状态订单数量和金额汇总
- 超时未确认 PO 自动提醒

#### 交期承诺跟踪
- 供应商承诺交期（Confirmed ETD）vs 实际交期（Actual ATA）记录
- 逾期分级预警：超期 1/3/7 天
- 交期准确率统计（按供应商、品类维度）

#### 到货质检跟踪
- IQC 检验结果回传看板
- 异常批次自动触发处理（退货/让步接收/挑选使用）
- 质量扣款自动计算

### 7. 🧾 对账与付款管理
#### 应付账款日历
- 按供应商排列应付账款，账期一目了然
- 预计付款日气泡图（本周/本月/下月）
- 超期未付自动标红

#### 付款优先级建议
- 综合评分：账期 + 折扣 + 供应商战略等级 + 合作关系
- 现金折扣自动识别（如 2/10 Net 30）
- 建议"先付谁、后付谁"，最大化资金效率

#### 三单匹配对账
- PO（采购订单）vs GRN（入库单）vs Invoice（发票）三方自动匹配
- 差异项标记：价格差异、数量差异、尚未入库
- 异常处理：差异原因记录 + 跟进责任人

### 8. 📡 行业情报与市场监控
#### 元器件价格指数
- 核心品类（MLCC、MOSFET、MCU、PCB、连接器、存储）价格走势
- 月/季/年度涨跌幅统计
- 与历史高低点对比，判断价格水位

#### 交期行情
- 常见料号标准交期 vs 当前市场交期
- 交期延长预警（如 MLCC 标准 8 周→当前 14 周）
- 紧缺料号榜单 + 推荐替代方案

#### 替代料推荐
- 紧缺/涨价时自动推荐兼容替代料号
- 替代等级：Pin-to-Pin / 功能兼容 / 需改板
- 替代料导入建议（测试验证周期预估）

### 9. 📊 KPI 仪表盘
#### 关键指标看板
- **降本达成率**：实际降本 / 年度目标 × 100%
- **OTD 准时交付率**：按时到货 / 总订单 × 100%
- **质量 PPM**：来料不良数 / 总来料数 × 1,000,000
- **供应商优化率**：年初供应商数 / 当前供应商数
- **库存周转天数**：库存金额 / 日均消耗金额
- **采购费用率**：运营费用 / 总采购金额 × 100%

#### 目标 vs 实际对比
- 月度/季度/年度进度条
- 红黄绿预警：<80% 红 / 80-95% 黄 / >95% 绿
- 同比上月 ±% 趋势箭头

#### 月度自评报告
- 一键生成采购月度述职摘要
- 关键成果 + 未达标分析 + 下月行动计划
- 老板版（只看数字）/ 详细版（含过程分析）

### 10. 🤖 日常效率工具
#### 邮件模板库
- 高频场景一键生成邮件：
  - 📩 询价函（BQ）：附 BOM + 目标价 + 交期要求
  - ⏰ 催货函：PO 号 + 逾期天数 + 最新交期
  - 🔴 品质投诉函（SCAR）：不良批次 + 不良率 + 8D 报告要求
  - 💰 议价函：市场行情 + 目标价 + 年度采购量筹码
  - 📋 季度供应商 review 邀约 + 议程
  - 🚫 供应商淘汰通知：原因 + 最后批次 + 交接安排

#### 会议纪要助手
- 供应商拜访模板：时间/地点/参与人/议题/决议/跟进项
- QBR 季度回顾模板：绩效回顾/问题/下季计划
- 谈判准备单：我方目标/底线/筹码/对方可能诉求

#### 采购日历
- 年度关键节点提醒：合同到期(提前60天)、审核计划、行业展会、年度降价谈判(Q4)、预算编制(Q3)
- 支持自定义事件

#### 知识沉淀
- 谈判案例库：背景+策略+结果+复盘
- 异常处理 SOP：缺料、品质、交期延误标准流程
- 采购百科：元器件基础、行业术语、验货标准
- 新人 onboarding checklist

### 11. 📋 自动报告
- 一键生成采购仪表盘（HTML）
- 异常自动预警：价格波动、交期延迟、库存告急
- 每月 1 日自动推送采购简报
- 支持导出 PDF / Excel

## 输入格式
- 支持 CSV / Excel 导入
- 自动识别字段：料号、品类、单价、用量、供应商、交货期、PO号、发票号等
- 内置电子元器件品类模板（IC / 被动件 / PCB / 连接器 / 结构件 / 包材）

## 输出格式
- 交互式 HTML 仪表盘
- 可导出 PDF / Excel 报告
- 邮件模板一键复制
- 数据表格 + 可视化图表 + 文字分析建议
