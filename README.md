# 采购智控助手 · Procurement Smart

电子元器件采购全流程管理工具包，覆盖供应商管理、成本分析、需求预测、PO 执行、对账付款、行业情报、KPI 仪表盘等采购全链路。

## 文件清单

| 文件 | 说明 |
|---|---|
| `SKILL.md` | Cowork 技能定义文件，完整的采购助手指令集 |
| `dashboard-full-suite.html` | 全功能仪表盘（供应商管理 + 五种成本分析） |
| `dashboard-cost-analysis.html` | 成本分析专题仪表盘（TCO/ABC/VA-VE/BOM/Target Costing） |
| `kpi-dashboard.html` | KPI 仪表盘（6 大指标 + 趋势 + 月度报告） |
| `monthly-report-june2026.pptx` | 向上汇报 PPT 模板（7 页完整报告） |

## 核心功能

- **🏭 供应商管理** — 五维加权评分（报价/品质/交期/服务/可持续供货）、全生命周期管理、绩效趋势追踪、合同资质预警
- **💰 成本分析** — TCO 总拥有成本、ABC 作业成本法、VA/VE 价值分析、BOM 成本拆解、Target Costing 目标成本法
- **📈 需求预测** — 移动平均/指数平滑/简易 ML 三种模型、季节性分析、安全库存建议
- **📦 库存健康度** — 库存水位 vs 安全水位 vs 预警线、缺料风险标注、自动补货建议
- **🔎 寻源引入** — 智能比价、资质预审、样品承认书管理
- **📄 PO 管理** — PO 全链路看板、交期跟踪、到货质检
- **🧾 对账付款** — 应付账款日历、付款优先级建议、三单匹配
- **📡 行业情报** — 价格指数追踪、交期行情分析、替代料推荐
- **📊 KPI 仪表盘** — 6 大指标看板、目标 vs 实际对比、月度自评报告
- **🤖 效率工具** — 邮件模板库、会议纪要助手、采购日历、知识沉淀

## 使用方式

所有 HTML 文件直接用浏览器打开即可交互使用。PPT 文件可用 PowerPoint / Keynote / WPS 打开编辑。

SKILL.md 可直接导入 Cowork 平台使用。
